# Minecraft基礎知識

## 1. 最初の目標
- まずは木を切って作業台を作ることが最優先。
- 作業台で木のツルハシを作る。
- 木のツルハシで石を掘り、石のツール（ツルハシ、斧、剣、シャベル）一式を作る。石のツールは木のツールよりずっと効率が良い。

## 2. サバイバルの基本
- **食料確保:** 周囲の動物（牛、豚、鶏）を倒して肉を手に入れる。かまどで焼いて食べると満腹度の回復量が上がる。
- **安全確保:** 夜になると敵対的なモンスターが出現する。最初の夜は、土を積んで仮の壁を作るか、山の側面に穴を掘ってやり過ごすのが安全。松明（石炭＋棒）を置くと、その周辺にモンスターが湧かなくなる。
- **ベッド:** 羊毛3つと木材3つでベッドを作れる。ベッドで寝ると、夜をスキップでき、リスポーン地点を更新できる。

## 3. 資源の優先順位
1.  **木:** すべての基本。常にインベントリにストックしておく。
2.  **石:** 基本的なツールや、かまどを作るのに必要。
3.  **石炭:** 松明や、かまどの燃料として非常に重要。地表近くの山肌や、洞窟の入り口でよく見つかる。
4.  **鉄:** 石炭の次に見つけたい重要な鉱石。鉄のツール、バケツ、防具など、作れるものの幅が大きく広がる。少し地下に潜った洞窟の壁際を探すと良い。

## 4. クラフトレシピの基本
- **作業台:** 原木 x1 -> 木材 x4, 木材 x4 -> 作業台
- **棒:** 木材 x2 (縦に並べる) -> 棒 x4
- **松明:** 石炭 x1 + 棒 x1 -> 松明 x4
- **木のツルハシ:** 木材 x3 (上の段) + 棒 x2 (中央)
- **石のツルハシ:** 丸石 x3 (上の段) + 棒 x2 (中央)
- **かまど:** 丸石 x8 (中央を空けて囲む)
- **チェスト:** 木材 x8 (中央を空けて囲む)
